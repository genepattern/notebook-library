Use GenePattern Notebook

Easily create and run a wide range of genomic analyses within shareable electronic notebooks

  • Our online workspace allows you to create and run notebooks using only your browser
  • Many common genomic and machine learning workflows are available in our Public Notebook Library and can be used with no programming required
  • Seamlessly integrate GenePattern analyses with R and Python
  • Based on and compatible with the Jupyter Notebook environment.
  • Share notebooks with collaborators
  • Publish a notebook and add it to your supplemental methods
  • Support computational research from inception through publication
  • See our Highlighted Publications
  • Access a variety of bioinformatics tools and resources from within a notebook
